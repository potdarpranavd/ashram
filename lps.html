Linear Power Supply

Designed linear power suppy to power different data acquisition systems and servo motors. Since most servo motors in my application used max 1.5A current at 5V, that circuit had fixed 5V, 2A with short-circuit protection. Some applications had servo motors as well as sensors attached to voltage source, so over-voltage protection was added to this circuit. In this case, the input was 230V AC mains.

In case of powering DC motors, a current booster circuit was also introducted for higher current requirements(upto 20A). In this case the input was 12V lithum ion battery.

I also designed battery charging circuit using linear power supply with current limiting resistor.
